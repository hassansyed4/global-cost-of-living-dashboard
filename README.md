# Cost of Living and Affordability Analysis Dashboard

An interactive Power BI dashboard designed to compare cities and countries based on income, expenses, and overall affordability. This project helps users identify where they can save more money and where living costs consume a larger share of income.

## Project Overview

This dashboard combines salary and living expense data to evaluate affordability across cities. Instead of looking at salary or costs separately, it brings them together into one decision-support tool.

The dashboard answers questions such as:
- Which cities are most affordable?
- Which cities are most expensive relative to salary?
- How much of monthly income is spent on living expenses?
- Which expense category takes the largest share of cost?
- Does a city create a monthly surplus or shortfall?

## Tools Used

- Power BI

## Dataset

The dataset contains city and country level cost-of-living metrics, including:
- Average Monthly Net Salary
- Housing Costs
- Food Costs
- Internet Costs
- Transport Costs
- Restaurant and grocery-related expenses
- Mortgage interest rate

All values were treated in USD for consistency.

## Dashboard Features

### KPI Cards
- Affordability Status
- Average Monthly Net Salary
- Total Monthly Cost
- Surplus or Shortfall
- Percentage of Income Spent

### Visuals
- Most Affordable Cities
- Most Expensive Cities
- Monthly Cost Breakdown by Category
- Monthly Expense Comparison Table

### Interactivity
- Country filter
- City filter
- Reset/Clear option
- Help guide panel
- Info panel with assumptions and project details

## Key Insights

- A city with a higher salary is not always more affordable.
- Housing is often the biggest contributor to total monthly cost.
- Percentage of income spent is a stronger affordability indicator than salary alone.
- Some cities create a financial shortfall even when salary appears reasonable.

## Business Value

This dashboard can support:
- Students exploring relocation options
- Job seekers comparing living conditions across cities
- Remote workers looking for cost-effective places to live
- Analysts studying income versus cost-of-living patterns
- Decision-makers presenting affordability insights in a visual way

## STAR Summary

### Situation
People often struggle to understand true affordability because salary and cost-of-living figures are usually viewed separately.

### Task
Build an interactive dashboard that compares cities based on salary, expenses, and affordability in a clear and user-friendly way.

### Action
Cleaned and structured the dataset, created affordability logic, built calculated metrics, and designed interactive Power BI visuals with filters, KPI cards, charts, and guide panels.

### Result
Developed a dashboard that transforms raw data into actionable affordability insights and supports better location-based financial decisions.
